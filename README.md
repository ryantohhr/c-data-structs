# Data Structures in C

This is my introductory project to C. In this project, I implement common data structures such as stack, queue, linked list and hash map in C.

My goal with this project was to learn the fundamentals of C and understand memory management and the usage of pointers.

### Some tools I picked up along the way:
- Makefile
- CMake
- Valgrind
